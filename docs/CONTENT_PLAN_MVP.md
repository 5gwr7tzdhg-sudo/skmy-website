# SKMY — контент-план MVP

## Цель

Сайт должен давать глухому иммигранту понятный первый ответ: что сделать, куда обратиться, какие документы подготовить и где проверить актуальные правила. Это план публикаций; реальные требования и официальные ссылки необходимо проверять перед написанием каждой инструкции.

## Единый шаблон статьи

Все статьи используют одну структуру в поле content:

    Краткое объяснение

    ## Шаги
    1. ...
    2. ...
    3. ...
    4. ...

    ## Необходимые документы
    - ...

    ## Официальные ссылки
    - ...

    ## FAQ
    Вопрос: ...
    Ответ: ...

    Вопрос: ...
    Ответ: ...

Приоритеты: P1 — публиковать в первой волне; P2 — важная статья полного MVP; P3 — расширение после MVP.
